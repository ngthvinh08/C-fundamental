# C Fundamentals

This repository contains my personal C programming practice projects. I originally completed these exercises as a freshman in university with limited programming experience. I am revisiting them now to strengthen my foundation in C/C++ while preparing for an interview at FPT Software.

## Overview

- Purpose: Review basic C concepts, improve coding fundamentals, and practice common programming problems.
- Topics included: arrays, pointers, functions, loops, recursion, strings, memory allocation, and branching.
- Context: The original work was done during my first year of university; the repository is being updated as part of ongoing study.

## Repository structure

- `array/` - array exercises and operations on array data.
- `basics/` - simple programs for area, perimeter, arithmetic, swapping values, and basic operations.
- `branch/` - examples of conditional statements like `if` and `switch`, checking odd numbers, and finding the largest number.
- `embedded C/` - examples in embedded C concepts.
- `function/` - function practice and code organization using reusable functions.
- `loop/` - loop exercises including `for`, `while`, Fibonacci, display output, and exponentiation.
- `memory allocate/` - dynamic memory examples using `malloc`.
- `pointer/` - pointer exercises, pointer-based swapping, and data copying.
- `recursion/` - recursive problems such as Fibonacci and the Tower of Hanoi.
- `string/` - string handling exercises, comparisons, uppercase conversion, and string operations.

## How to run

Use a C compiler such as `gcc` to compile and run each file:

```bash
gcc -o output.exe path/to/file.c
./output.exe
```

For Windows with MSYS2 UCRT64, for example:

```bash
C:/msys64/ucrt64/bin/gcc.exe -o bai1.exe array/bai1.c
./bai1.exe
```
